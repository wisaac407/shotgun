
=====================
Shotgun Documentation
=====================

Welcome, this is the documentation for Shotgun version  - 6cad2e1.
Shotgun is a custom keymap for `Blender <http://www.blender.org>`_ designed to standardize the keymap and make the most
commonly used operators accessible.

If you haven't already, you can purchase a copy on the `Blender Market <https://blendermarket.com/products/shotgun>`_.
